# Tree of Thought Solver



You are a rigorous problem solver. Break down the problem into
incremental reasoning steps, evaluate each intermediate state,
and find the most sound reasoning path.

Problem: A startup has 3 engineers and needs to ship an MVP in 6 weeks. They must choose one numbered option: 1 = A, a polished web app with limited features; 2 = B, a feature-rich prototype with rough UX; 3 = C, a CLI tool plus API with no frontend. Each engineer can only focus on one major component. How should they allocate work to maximize chances of getting paying customers? End with the numeric option number (1, 2, or 3).

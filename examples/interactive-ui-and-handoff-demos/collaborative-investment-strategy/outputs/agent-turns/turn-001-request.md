## 4. Risk Management & Rebalancing Plan

### Rebalancing Rules

Rebalance the portfolio:

- **Once per year**, preferably on a set date, or
- Whenever an asset class drifts more than **5 percentage points** from its target allocation

Example: if U.S. equities rise from 50% to 57%, rebalance by directing new contributions toward underweight assets or selling a portion if needed.

### Use Contributions First

Because the investor is contributing $3,000/month, rebalancing should usually be done by adjusting new contributions rather than selling existing holdings. This reduces taxes and transaction costs.

### Lump Sum vs. Dollar-Cost Averaging

Mathematically, investing the $150,000 lump sum immediately has historically produced better outcomes more often than waiting. However, if volatility is a concern, a reasonable compromise is to invest the lump sum over **3 to 6 months**.

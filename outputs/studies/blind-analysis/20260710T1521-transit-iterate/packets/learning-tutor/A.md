## Source material

# variant prose Linear Algebra Tutor

Create a chat prompt for an adaptive linear algebra tutor.

The tutor should teach a motivated returning beginner who remembers algebra but
has weak geometric intuition. The tutor should not lecture all at once. It
should begin by probing the learner's current mental model, ask one focused
question at a time, and adapt based on the answer.

The lesson should teach vectors, spans, matrices as transformations,
eigenvectors, and eigenvalues through intuition first, precise formulation
second, worked examples third, and edge cases last. It should use the idea of a
matrix as a machine that moves space, but it must also state where that analogy
breaks.

The tutor should diagnose misconceptions. It should separate notation confusion,
arithmetic slips, missing prerequisites, false analogies, and conceptual
inversions. When a learner gives a wrong answer, the tutor should ask for the
reasoning path when needed, explain why the misconception is tempting, use a
small contrast case, and immediately give a repair exercise.

The tutor should include a mastery loop. It should define what the learner should
be able to do, provide recognition, application, and transfer practice, give
specific feedback, simplify after repeated failures, and include a delayed
review prompt.

The output should be a pasteable tutor prompt with:

1. tutor role and teaching principles;
2. first interaction;
3. adaptive question sequence;
4. misconception diagnosis rules;
5. practice ladder;
6. feedback rules;
7. final mastery check.
